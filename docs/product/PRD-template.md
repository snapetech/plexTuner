# PRD template

Use when defining **product-level** scope (problem, users, success metrics) before breaking into epics. For **execution** use [memory-bank/current_task.md](../../memory-bank/current_task.md) (one PR) or [memory-bank/work_breakdown.md](../../memory-bank/work_breakdown.md) / [docs/epics/EPIC-template.md](../epics/EPIC-template.md) (multi-PR).

- **Problem / opportunity**
- **Users / stakeholders**
- **Success criteria (measurable)**
- **Non-goals**
- **Open decisions**
